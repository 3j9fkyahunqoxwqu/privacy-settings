Enable or Disable Policy Rules1:

  • Font (@font-face) on allOn
  • Image on allOn
  • Media (audio and video) on allOn
  • Object (plugin-handled content) on allOn
  • Script (JavaScript) on allOn
  • Stylesheet (CSS) on allOn
  • Sub Document (iframes and frames) on allOn
  • XMLHttpRequest on allOn
  • Plugins on allOn
  • WebSocket on allOn
  • Ping on allOn


1: By enabling a rule, websites will NOT be able to access that particular resource anymore. However, you can still allow access of some resources using custom filtering rules.
2, 3: Disable these options to increase performance.
4: Instead of temporary disabling all rules to be able to browse a website that is not loading while policy rules are enabled, it is recommended to turn off policy rules permanently in the private browsing mode.
